# Outpatient-Waiting-Time-Analysis
Analysed 970K+ outpatient appointment and consultation records extracted from Oracle databases using SQL and Python to identify operational bottlenecks affecting waiting time and support hospital workflow optimisation.

## Note
The original dataset contains confidential hospital records and cannot be released publicly.
A small anonymised sample is provided for demonstration purposes.

## Dataset
- Source: Oracle Database
- Size: 973,835 outpatient records
- Data Type: Appointment and consultation records
- Privacy: Original data cannot be released due to confidentiality.

## Project Workflow
Oracle -> SQL -> Cleaning -> EDA -> Business KPI -> Recommendations

## Key Outcomes
- Analysed over 970,000 real-world outpatient records.
- Identified abnormal waiting-time patterns.
- Developed an end-to-end data cleaning and analytics workflow.
- Generated business recommendations to improve outpatient operational efficiency.
